### v2rayN-5.39旧界面版

下载net4.8环境：https://dotnet.microsoft.com/zh-cn/download/dotnet-framework/net48

`SDK`包和`运行时`都要下载安装。

下载本项目，然后`cd v2rayN`

链接路径`v2rayN\Global.cs`

按钮文字路径`v2rayN\Forms\MainForm.zh-Hans.resx`



构建命令
```
dotnet build --configuration Release
```

---


清理
```
dotnet clean
```
构建
```
dotnet build
```
